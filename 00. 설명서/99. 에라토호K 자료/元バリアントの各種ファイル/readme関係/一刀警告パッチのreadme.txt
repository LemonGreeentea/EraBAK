1. 단칼에 의한다ＮＴＲ에의 경고문 표시를 추가
　
  단칼에 손을 대어지고 있는 캐릭터의 반하고 번이 일정 이상 높아졌을 때에 경고문이 표시되게 됩니다

  경고는 2단계가 되어 있어, 각각 한 번만 표시하게 되어 있습니다



2. 단칼에의 연모·연인 취득시라고 경고문의 색변경

  단칼에의 연모·연인 취득시라고 경고문의 색을 바꾸어 확인하기 쉽게 합니다



3. CFLAG의 추가

  미사용의 『CFLAG:90』를 경고문 표시가 매회 표시되지 않기 위한 플래그 사용했습니다

  『era 연희[戀姬] 변수표』에 『90:단칼에의 반하고 번의 경고 플래그』를 추가 기입이라고 있습니다
