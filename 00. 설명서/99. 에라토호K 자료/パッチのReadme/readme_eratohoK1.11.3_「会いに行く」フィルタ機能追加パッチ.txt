eratohoK1. 11.3_ 「만나러간다」필터 기능 추가 패치


예 1:아이 캐릭터 전원을 가능한 한 육성하고 싶지만, 란 댐 캐릭터 있어로 했기 때문에 아이 캐릭터와 뒤섞이고 있어 만나는 것이 귀찮음! 나누어 표시할 수 있으면 좋은데!
예 2:특수 세력으로 플레이 해 함락 인원수를 벌고 싶은 것이지만, 「모」나 「옷」보다 「타」의 표시가 우선되기 때문에 함락이 끝난 상태인가 어떤가의 판별이 귀찮음! 나누어 표시할 수 있으면 좋은데!

이런 이유로, 어느 특정의 조건을 채우는 캐릭터만 「만나러간다」에 표시되도록(듯이) 바꿀 수가 있습니다.
조건은 「필터 설정」으로 선택해 주세요. 전항목을{○마셔}나{×마셔}로 하면, 1건도 표시되지 않습니다.


지금은 「필터 OFF」 「필터 1」 「필터 2」의 3개를 차례로 바꾸고 할 수 있습니다만, 4개나 5개로 하고 싶은 경우는
SHOP_LIFE11_만나러간다_필터. ERH
노
#DIM SAVEDATA SHOP_LIFE_11_FILTER, 3, 10
의 3을 4나 5로 변경해 주세요.

필터 선택 항목을 늘리고 싶은 경우는,
SHOP_LIFE11_만나러간다_필터. ERB
노
@SHOP_LIFE_11_GET_FILTER_ITEM(CHARA, IDX, NAM)
그래서, CASE 9이후를 추가해 주세요.
10항목으로 부족한 경우는
SHOP_LIFE11_만나러간다_필터. ERH
노
#DIM SAVEDATA SHOP_LIFE_11_FILTER, 3, 10
의 10을 20이나 30 등에 변경해 주세요.

필터에 관한 변수의 내용이 세이브됩니다. 세이브 데이터에 영향을 주고 싶지 않은 경우는
SHOP_LIFE11_만나러간다_필터. ERH
로부터
SAVEDATA
의 키워드를 삭제해 주세요. 2개소.



처음은,{ 양쪽 모두 }  {○마셔}  {×마셔} 만이 아니어서
캐릭터 종별    { 모두 }  { 고정 }  { 범용 }  { 아이 }  { 특수 }
같은 것도 만들까하고 생각했지만, 처리가 복잡하게 될 것 같은 것으로 그만두었습니다.



라이센스 같은 것
SHOP_LIFE11_만나러간다. ERB
하, 아주 조금 고쳐 쓴 것 뿐인 것으로, 변경전의 파일과 동일시 해 주세요. 나의 영향은 없다는 것이라고.

SHOP_LIFE11_만나러간다_필터. ERH
하, 라이센스 운운을 신경쓰는 것 같은 내용이나 양이 아닙니다.

SHOP_LIFE11_만나러간다_필터. ERB
의 개변이나 유용은, 「era로 이용한다면 자유」 「era 이외로 이용한다면 불가」로 합니다. 굉장한 물건도 아닙니다만 일단.
「만나러간다」이외에도 이용하는 경우는, 변수명이나 함수명을 변경해야할 것인가도 모릅니다.
